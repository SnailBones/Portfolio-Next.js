# What Trees Were

_What Trees Were_ began as a final project for my Computer Graphics class and evolved into a creative tool for all your tree-generation needs. Tweak the ruleset to create the tree of your dreams, or use an advanced mode allowing for interpolation between two rulesets in one tree.

Written in Java.

Download on [Itch.io](https://ailanthus.itch.io/whattreeswere) or check out the source code on [Github](https://github.com/SnailBones/Tree3D).
